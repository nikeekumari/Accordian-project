FAQs in HTML-HYPERTEXT MARKUP LANGUAGE

What is html?

HyperText Markup Language (HTML) is a basic scripting language that defines the structure and meaning of web content.

Who developed html?

Tim Berners-Lee

What is HTML used for?

HyperText Markup Language (HTML) is a basic scripting language that defines the structure and meaning of web content. It's used by web browsers to render pages on the World Wide Web. HTML is made up of elements that tell the browser how to display content.
